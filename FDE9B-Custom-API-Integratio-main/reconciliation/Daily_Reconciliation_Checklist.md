# Daily Reconciliation Checklist

## Daily Activities

- Verify successful SAP data extraction.
- Verify successful API execution.
- Compare source and destination record counts.
- Compare financial totals.
- Review failed transactions.
- Review duplicate records.
- Validate mandatory fields.
- Review reconciliation report.
- Escalate unresolved discrepancies.
- Archive reconciliation logs.

---

## Completion Criteria

The reconciliation process is complete when:

- All records are reconciled.
- Financial totals match.
- Exceptions are resolved or documented.
- Reports are archived.
